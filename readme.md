# dkdc

***bookmarks in your terminal***

## installation

```bash
go install dkdc
```

## usage

### config

Open the config file with:

```bash
dkdc --config
```

Or:

```bash
dkdc -c
```

## open

Open something with:

```bash
dkdc thing
```

Or:

```bash
dkdc alias
```

Or:

```bash
dkdc a
```
